# Changelog

All notable changes to this project will be documented in this file.

## [0.2.1] - 2026-07-16

### Changed

- Refined the Korean PyPI README with clearer first-use, agent-installation, dataset-switching, and indexing guidance.

## [0.2.0] - 2026-07-15

### Added

- Added one-command Codex and Claude Code MCP host setup with safe conflict handling.
- Added persistent active dataset selection and MCP tools for changing or resetting it.
- Added agent-first PyPI installation guidance and automatic first indexing when documents exist.

## [0.1.1] - 2026-07-12

### Changed

- Rewrote the Korean README as a beginner-friendly, task-oriented setup guide.
- Moved the project links to `jaypakdevkr/kb-mcp-server`.
- Simplified the default `~/Desktop/dataset` workflow and troubleshooting guidance.

## [0.1.0] - 2026-07-12

### Added

- Local HWP/HWPX parsing, element-aware chunking, multilingual E5 embeddings, and FAISS search.
- Safe FAISS and JSON persistence without pickle deserialization.
- Explicit index status/sync CLI and MCP tools for Codex and Claude Code.
- Korean-first setup and operations guide.
